Directions on how to use our API:

Explore our API to get more information on certain products in our inventory. You can search for items based on distance or see all of our items listed by price. See the (sample) docs below for specific routes. Thanks!

Route Description Example
api/sortByCreationDateAsc Returns an array of all items, sorted by their creation date, in ascending order (oldest items to newest items) [{createdAt: "2015-03-26T23:00:18.800Z"},{createdAt: "2015-05-26T23:00:18.800Z"}]
api/sortByCreationDateDesc Returns an array of all items, sorted by their creation date, in descending order (newest items to oldest items) [{createdAt: "2015-05-26T23:00:18.800Z"},{createdAt: "2015-03-26T23:00:18.800Z"}]
api/price/Asc Returns an array of all items, sorted by their price, in ascending order (cheapest item to most expensive item), "-1" means the item has not been priced and will be listed first [{price: -1}, {price: 10}, {price: 35}]
api/price/Desc Returns an array of all items, sorted by their price, in descending order (most expensive item to cheapest item), "-1" means the item has not been priced and will be listed last [{price: 35}, {price: 10}, {price: -1}]
api/items/:userId Returns an array of all items that are listed by a particular user based on userId '/items/:123' returns [x,y,z] where x, y, and z are all the items listed by user 123
api/item/:id Returns the item with the matching id '/item/:789' returns {id: 789, description: "houseware"}
api/items/:lat/:lon/:distance Returns an array of all items that are within a certain distance of the input location via latitude and longitude coordinates. The distance is measured in miles by default. '/items/a/b/c' returns [item1,item2,item3], where item1, item2, and item3 are all items within c miles of the location coordinates a and b
